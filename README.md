# tcc-controle-financeiro-pessoal
Trabalho de Conclusão de Curso para Controle Financeiro Pessoal

O projeto está dividido em cliente e servidor.
No lado cliente está sendo usado o framework Angular.
O lado servidor está utilizando a linguagem de programação Java juntamento com o framework Spring.

Para rodar o projeto siga os seguintes passos:
tenha uma base de dados PostgreSQL com o nome: 'controle-financeiro' ou rode com o h2 ajustando a config do arquivo application.yml.
API: instale as dependencias do mavem e rode o projeto em sua IDE na porta 8080.
Front: instale as dependencias do projeto ´npm install´ e rode o projeto com ´ng serve´
